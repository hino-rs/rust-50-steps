# 華氏から摂氏へ

## 学習

- 関数を定義するには、`fn`を使います。
- `fn <関数名>(引数名1: 型1, 引数名2: 型2) -> 戻り値の型 {}`
- もちろん、引数や戻り値が無い関数も作れる。
  
- `println!()`の`"{}"`を、`"{:.n}"`とすると、`n`の数まで小数部を表示する。`println!("{:.1}", 3.14)` -> `3.1`

## 要件定義

華氏から摂氏へ変換するツールを作ってください。

**実行例**:

```bash
華氏を入力してください
75.0 # 入力
75.00°F → 23.87°C
```

- エラーハンドリングは`expect`で適切なメッセージを出力させてください。
- `convert_to_celsius`関数を作ってください。

シグネチャ:

```rust
fn convert_to_celsius(f: f64) -> f64 {}
```

華氏→摂氏の公式:

$$摂氏 = (華氏 - 32) * 0.555$$
