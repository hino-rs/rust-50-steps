# パスワード生成

## 学習

- イテレータのメソッドに`nth(n)`があります。これは`n`個めの要素を取得する便利なものですが、手前から数えるため、イテレータが膨大だとボトルネックになります。
- イテレータのメソッドに`chain()`があります。イテレータを繋げることができます。
- イテレータをのメソッドに`len()`があります。要素数を取得できます。
- `0..=10`は`0`~`10`でしたが、`'a'..='z'`と`'A'..='Z'`も使えます。

## 要件定義

- パスワード生成器を作ってください。
- パスワードはa-z, A-Z, 1~9の範囲で、コマンドライン引数の値の数を文字数にします。
- `for`の使用は禁止です。

**実行例**:

```bash
cargo run -p password_generator -- 16
NxUF4ekYRhwalgid
```
